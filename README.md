# dorevia_vault_connector
Assurer la connexion sécurisée et automatisée entre Odoo et le service Dorevia Vault afin que :  chaque document validé dans Odoo (facture, avoir, devis, etc.) soit envoyé dans le Vault,  son empreinte cryptographique (SHA256) et ses métadonnées soient archivées,  et que l’état de vaultage soit visible et traçable dans Odoo.
